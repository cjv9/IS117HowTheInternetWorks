Domain Names and ICANN

Internet Corporation for Assigned Names and Numbers (ICANN)

ICANN is a nonprofit that maintains and manages domain names and numbers of the internet, which ensures its integrity and stability.

ICANN performs both physical and virtual maintenance of root name server sites, and of policies surrounding domain, IPV4, and IPV6 addresses.

Domain names are letter combinations by which sites are identified as. While internet sites are only accessible by ip addresses, they're not very user friendly, so domain names are what are searched for in the user end to send a request for the sites IP address.

ICANN manages domains, while sites like GoDaddy market them out to users, paying a percentage to ICANN, and are paid by users to use domains.

We use domain names to easily identify and request for websites, and ICANN provides the internet scope with maintenance and stability.
